This project builds a machine learning model to predict wine quality based on chemical properties such as acidity, pH, alcohol, and sulphates. The dataset is explored and preprocessed, including converting quality into binary classes for classification. The data is split into training and testing sets, and a Random Forest model is trained to make predictions. Model performance is evaluated using accuracy and classification metrics. A correlation heatmap is used to understand relationships between features and wine quality. The project demonstrates a complete machine learning workflow using Python, Pandas, and Scikit-learn.
